why watkit?

learn low-level

published wat = hackable webassembly. decompiled wasm binaries are hard for experienced devs to use, let alone for beginners to learn from. import from a readable registry instead!

optimize performance

more people writing wat by hand means more optimized wasm code and a faster web for everyone.

build a community

share and reuse "standard" code in a fully modular ecosystem with versioned, recursive dependency support. focus on innovation, not rewriting common patterns.
